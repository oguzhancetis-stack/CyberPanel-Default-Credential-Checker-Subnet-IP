CyberPanel Security Auditor

A lightweight tool to assess the security posture of CyberPanel instances within an authorized subnet scope.

🚀 Overview

This tool scans a given subnet to discover CyberPanel panels and identify potential security risks such as exposed services and weak authentication configurations.

Designed for system administrators, internal audits, and lab environments.

⚙️ Features
Subnet-based scanning (CIDR support)
Detects CyberPanel login panels
Identifies exposed services
Highlights weak authentication risks
Fast and efficient scanning
📦 Usage
python cyberlogger.py --subnet 192.168.1.0/24
